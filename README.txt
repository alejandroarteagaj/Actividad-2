#Actividad 2

Parte 1:

En este repositrio se encuentra la propuesta de rediseño del detector de neumonia, en este rediseño se plantearon tres scripts
donde cada uno de estos esta enefocado a una funcion especifica dentro del codigo.

UI: Solo gestiona la interacción con el usuario y no debe tener lógica que procese o vaya más allá de validaciones.

backend: Contiene una lógica que procesa datos de entrada y genera salidas y mensajes para la UI

Inference: Se encarga de procesar imágenes y retorna las predicciones

Para correr el sistema de forma local realizar los siguientes pasos:

1. Clonar el repositorio utilizando el siguiente link: https://github.com/alejandroarteagaj/Actividad-2.git

2. Entrar a la carpeta de Scripts

2. Una vez dentro de la carpeta descargar el modelo .h5 desde el siguiente link: https://drive.google.com/file/d/1IRctNklwe2Aa2dBRK32nYGtS7xtJ1CXr/view?usp=sharing

4. Ejecutar el Script referente a la interfaz de usuario UI.py

Nota: Dentro del repo clonado see encuentran las imagenes de prueba para el testing del modelo.
